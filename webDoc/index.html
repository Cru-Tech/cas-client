CAS Java Client

version 2.1.0

This is the CAS Java Client 2.1.0. It evolved from the CAS Java Client alpha release which evolved from the official beta CASClient release to include Commons logging, JUnit tests, exposure of additional information about successful authentications, a filter that does only validation (no redirects), etc.

Contact: Drew Mazurek (drew dot mazurek at yale dot edu), CAS project maintainer, Yale University

Download

This Java CAS Client release is distributed as a .zip file which contains this documentation, source code, testcases and their results, javadocs, and, of course, a casclient.jar file.
